韋諾之戰 1.18
龍族入侵更新

2024 年 3 月開發團隊在此隆重推出 1.18 版本的韋諾之戰!

此版本增加許多新的內容，並對現有的內容進行改善，包括新的龍族戰役、最古老親皇派戰役的全面重製、圖形效能的大幅改善，還有針對多人遊戲的平衡調整、成就系統、編輯器擴充，以及其他更多內容！等您親自來體驗，透過下方內容來查看更多細節。

本次更新內容

針對玩家的更新

韋諾之戰的程式渲染已全面改造，性能大幅提升，根據我們的測試，每秒傳輸幀數提升將近 60%。這個表現取決您的電腦性能，但無論如何都會使遊戲在各個領域的體驗更完整和更加一致。

預設時代的所有派系都經歷重大更新，以確保一對一時更加平衡，並確保戰役使用新的可變召喚成本機制。高級單位也進行各種調整，更好地平衡了英雄時代以及高級單位普遍存在的生存章節。已新增蜥蜴先知和蜥蜴預言者的第三階段升級，而且半獸人獵殺者將可升級成半獸人夜刃。

戰役

此次新增一個新戰役《命運之風》！首次在韋諾之戰裡加入龍族戰役，展現他們為生存而戰，逃離沉沒的島嶼家園後重返韋諾大陸的動盪歷程。

《東部入侵》 — 在韋諾之戰裡首次加入的親皇派戰役 — 目前已經全面重新製作，角色更加深入，情節更加豐富，並且搭配更加動態的人工智慧對手，增加了更多的章節變化。同時也解決了與其他戰役之間的一些故事衝突。

最後，在《烈日當空》 戰役中的昆諾種族已經進行了大幅度的改版，故事更加有趣 — 英雄單位升級時可以獲得獨特的獎勵，而普通單位能在升級時擁有新技能和攻擊時的新武器效果，這將成為韋諾之戰裡戰役中獨一無二的體驗。

多人遊戲

現在玩家可以查看自己先前的遊戲歷史紀錄，也可以點選線上多人遊戲大廳中的「對戰歷史」來搜索其他玩家的遊戲歷史紀錄。歷史紀錄會顯示有關遊戲本身、遊戲中的玩家、使用的時代、啟用的任何修改，並讓您下載遊戲的重播來觀看。此外，還能點選線上戰情室來查看更多細節。

現在，自動進行的遊戲（例如 Afterlife）與其他玩家主持的自訂遊戲已被區分開來。玩家主持的遊戲將顯示在「一般大廳」中，而自動進行的遊戲則會顯示在「配對」中。為了幫助玩家回報其他有問題的玩家或是遊戲本身的問題，在/report中有一個對話框，玩家可以在對話框內輸入發現的問題。此外，為了能在網路出問題而導致伺服器斷線時做出更好的應對措施，韋諾之戰現在會自動檢測您的連線是否中斷，並主動告知您此訊息。

為鼓勵玩家佔領中央要塞，山丘之王地圖已經更新為：無論玩家佔領山頂的哪一方，都能獲得可觀的額外收入，而且第一個佔領者還可獲得一個鉅額寶箱。

最後作為一個驚喜，伊薩爾交叉口上的瀑布現在已經加入了動畫效果。

針對創作者們的更新

韋諾之戰現在支援內建成就系統！這些成就並非 Steam 成就，也因此並不限於基本遊戲中預定的成就集 — 模組可以定義自己的成就，甚至可以向其他模組或主線遊戲中添加額外的成就。在這個初始版本中，已為教學戰役、墮入黑暗、東部入侵、自由以及多人地圖黑暗預測、迷霧之島、新土地和團隊生存實現了成就系統。

關於您自製模組的發佈，韋諾之戰現在允許使用您的論壇使用者名稱和密碼來上傳模組。這樣一來，模組的密碼將儲存在加密文件中，而不是以純文本形式儲存在您的_server.pbl中，這將更具更安全性，但也意味者，如果您忘記密碼，必須要使用論壇的密碼恢復功能來重新取得存取許可。此外，對於多人合作開發的模組，可以讓被指定的用戶對該模組進行更新。在上傳模組前，必須先驗證您的_server.pbl，就能在上傳前就知道是否存在任何問題，而不是之後才發現。

章節編輯器進行重大升級：現在不僅能理解單一章節，還可以讓您製作新模組的結構，也提供在製作_server.pbl時的使用者介面。這意味著現在您可以在韋諾之戰裡製作一個模組、該模組的_server.pbl及該模組的劇情，並上傳該模組供其他玩家使用。

地圖編輯器也得到重大改進：現在可以加入懸崖和山谷，為您的地圖增加不少的細節。除此之外，還能新增各種新的地形和生物，從柵欄牆、貝殼裝飾和海藻地形到火蟻、巨型跳蛛和巨鷹都有。

許多精靈單位的圖像得到改善，並新增各種動畫，主要是保皇派、半獸人和巨魔，但也包括一些精靈和不死族的單位。

最後，幾乎所有韋諾之戰定義的 Lua 函數現在都有 LuaCATS(Lua Comment And Type System) 支援，在輸入時會自動填寫，並在滑鼠停在上方時提供文檔。這主要是為了 Visual Studio Code 而編寫，但應適用於任何具有 LuaCATS 模組的編輯器。

貢獻

許多來自不同背景的國際志願者不間斷地對韋諾之戰進行改進和擴展。自 2003 年 7 月創立以來，數百人在遊戲的各個領域做出了貢獻，從官方發布的主要內容到數百個玩家製作的模組，都可以透過我們的模組伺服器來下載。無論您需要幫忙來打贏特定的章節，或是在您的自定地圖上添加某個功能，韋諾之戰都擁有一個龐大的玩家和創作者社區，隨時為您提供協助。

除了創作您自己的模組之外，還有其他許多方式可以讓韋諾變得更好：

  • 貢獻編碼
  • 為主遊戲製作美術圖像或自製模組
  • 翻譯遊戲成您的語言
  • 製作新的音樂曲目和音效
  • 在社群媒體上宣傳韋諾之戰的訊息！
